# Databricks LLM Pipeline with MLflow + Vector Search

## 📌 Overview
An end‑to‑end LLM application built entirely on Databricks.  
Includes data ingestion, embeddings, vector search, MLflow tracking, and model serving.

## 🚀 Features
- PySpark-based data ingestion  
- Embeddings generation using Databricks Model Serving  
- Vector search using Databricks Vector Search  
- MLflow experiment tracking + model registry  
- LLM-powered Q&A or summarization  
- Production endpoint via Databricks Model Serving

## 🔧 Setup Instructions
1. Create Databricks workspace + cluster.  
2. Ingest dataset using PySpark.  
3. Generate embeddings via Model Serving.  
4. Build vector index in Databricks Vector Search.  
5. Track experiments using MLflow.  
6. Deploy model to Model Serving.  
7. Build a simple UI or API.
